🔑 Getting API Keys

Need API keys? Get them here:

  • MiniMax: https://api.minimaxi.com
  • DeepSeek: https://platform.deepseek.com

💡 Tip: If you leave API keys empty, the system will show common models as fallback options.

📝 Agent Roles Configuration

Configure agent roles:

  • Set display name for each role
  • Select provider for each role
  • Choose model from available options
  • Customize system prompts

💡 Tip: Models can be fetched directly from providers using the "Get Models" button.

⚙️ Pipeline Parameters

Configure pipeline parameters:

  • Max steps for each pipeline phase
  • Temperature settings
  • Max tokens limits

💡 Tip: Adjust these parameters to control the behavior of your AI pipeline.

🔒 Approval Settings

Configure auto-approval paths:

  • Set working paths for auto-approval
  • Remember approved files

💡 Tip: Use this to automatically approve writes to specific paths.

📢 Notifications

Configure notification channels:

  • Add Feishu/DingTalk/WeChat/Telegram channels
  • Choose notification modes
  • Test channel connectivity

💡 Tip: Use simple mode for key events, complex mode for real-time updates.

📦 Version Control

Configure version control:

  • Set up GitHub/Gitee/GitLab accounts
  • Create repositories
  • Push/pull projects

💡 Tip: Use tokens with 'repo' scope for full repository access.
